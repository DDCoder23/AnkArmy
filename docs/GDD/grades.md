# Grades

Version : 0.1
Statut : En cours de conception

---

# 1. Objectif

Les grades représentent la progression officielle de la carrière militaire du joueur.

Ils matérialisent l'expérience, la discipline et l'engagement du joueur au sein d'AnkArmy.

Une promotion n'est jamais uniquement esthétique : elle débloque toujours de nouvelles fonctionnalités.

---

# 2. Principes

Le système de grades repose sur les règles suivantes :

* les grades sont permanents ;
* un grade ne peut jamais être perdu ;
* chaque promotion est méritée ;
* les promotions sont accordées automatiquement par le Commandement ;
* les grades reflètent la progression globale du joueur.

---

# 3. Structure générale

La carrière est divisée en cinq grandes phases.

## Phase I — Formation

Le joueur découvre AnkArmy et apprend les bases.

Grades :

* Recrue
* Aviateur confirmé
* Caporal
* Caporal-chef

Objectifs :

* comprendre le fonctionnement du système ;
* passer du statut de recrue à celui d'aviateur confirmé ;
* développer une routine de révision.

---

## Phase II — Sous-officiers

Le joueur devient régulier.

Grades :

* Sergent
* Sergent-chef
* Adjudant
* Adjudant-chef
* Major

Objectifs :

* renforcer la discipline ;
* améliorer la précision ;
* débloquer les premières missions avancées.

---

## Phase III — Officiers

Le joueur maîtrise pleinement AnkArmy.

Grades :

* Aspirant
* Sous-lieutenant
* Lieutenant
* Capitaine
* Commandant

Objectifs :

* optimiser les performances ;
* accéder à des opérations plus complexes ;
* développer une carrière exemplaire.

---

## Phase IV — Officiers supérieurs

Le joueur atteint un haut niveau d'expérience.

Grades :

* Lieutenant-colonel
* Colonel

Objectifs :

* maintenir une excellente régularité ;
* accomplir des campagnes de longue durée.

---

## Phase V — Haut Commandement

Le joueur entre dans les plus hauts niveaux de la hiérarchie.

Grades :

* Général de brigade aérienne
* Général de division aérienne
* Général de corps aérien
* Général d'armée aérienne

Objectifs :

* atteindre l'excellence ;
* terminer une carrière militaire exceptionnelle.

---

# 4. Conditions de promotion

Une promotion dépend de plusieurs critères.

Le Commandement peut prendre en compte :

* l'expérience (XP) ;
* la discipline ;
* les missions accomplies ;
* les performances générales ;
* les éventuelles restrictions disciplinaires.

Les valeurs exactes sont définies dans le document `balancing.md`.

---

# 5. Déblocages

Chaque promotion peut débloquer :

* de nouveaux types de missions ;
* de nouveaux équipements ;
* des rapports plus détaillés ;
* des analyses statistiques avancées ;
* des décorations ;
* des campagnes spéciales.

Aucun grade ne doit être inutile.

---

# 6. Annonce de promotion

Chaque promotion donne lieu à un rapport officiel.

Exemple :

==================================================

PROMOTION OFFICIELLE

Après examen de votre dossier militaire,

le Commandement vous accorde le grade de

CAPORAL

Félicitations.

==================================================

---

# 7. Régression

Le joueur conserve définitivement ses grades.

En revanche, une baisse de discipline peut entraîner :

* la suspension temporaire de certains bonus ;
* des missions plus exigeantes ;
* des avertissements disciplinaires.

Le système privilégie toujours la progression plutôt que la punition.

---

# 8. Évolution future

Les grades pourront, dans les futures versions :

* débloquer des campagnes ;
* ouvrir de nouvelles branches de carrière ;
* donner accès à des responsabilités supplémentaires ;
* enrichir les rapports militaires.

---

# 9. Règle fondamentale

Chaque promotion doit procurer un sentiment d'accomplissement.

Le joueur doit avoir le sentiment de construire une véritable carrière militaire qui s'étend sur plusieurs mois ou plusieurs années.
