# 🪖 AnkArmy

## 🎯 Qu’est-ce que c’est ?

AnkArmy est un addon Anki qui transforme les révisions en jeu de progression militaire.

Chaque session de révision devient une mission.
Chaque carte est un ordre.
Chaque réussite te fait progresser dans la hiérarchie.

---

## 🚀 Objectif du projet

Rendre l’apprentissage :
- plus régulier
- plus motivant
- plus “game-like”

---

## 🧠 Concept

Tu commences comme recrue.

Ton but :
➡️ exécuter des ordres (cartes Anki)
➡️ gagner de l’expérience
➡️ monter en grade

---

## 🪖 Progression

- Recrue : début
- Soldat : régularité
- Caporal : discipline
- Sergent : performance
- Officier : maîtrise

---

## 🎮 Gameplay

Chaque session Anki devient :

📡 “Mission du jour”
- objectifs
- score
- bonus
